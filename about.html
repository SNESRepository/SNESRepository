About this Website

A digital curation and preservation website for Super Nintendo Entertainment System/Super Famicom ROM images.

Use the navigation bar at the top of the website to search for SNES ROM images that have been curated and preserved on this website. Each game has it's own page containing metadata and further information about the game.

The metadata fields for each SNES ROM image are the following:

  • Name - The name of the game/software in the SNES ROM image
  • Description - A short description of the game/software
  • Developer - Who originally developed the game/software
  • Release Date - When the game/software was originally released
  • File Size - The size of the ROM image
  • Checksum - An MD5 checksum to verify that the ROM image is clean and genuine

Curation process:

ROM images have be extracted (dumped) using a rom dumper. The rom dumper I have chosen to use for dumping the SNES ROM images from each cartridge is the Retrode 2. The Retrode 2 is competitively priced, accepts cartridges for SNES and Sega Genesis games (among others with adapters), and is easy to use, making it a clear choice for this project and potential future digital curation/preservation projects of fourth generation video game ROM images. It is worth noting that the stock SNES console restricts players from booting games published in regions outside of the US through a slightly different cartridge design and cartridge bay, but the Retrode 2 does not have these restrictions. Using the Retrode 2 is straightforward and simply connects to a PC, laptop, or Mac though a USB cable. Each cartridge was be placed into the Retrode 2’s cartridge bay, where it’s contents where then displayed on the computer the device is connected to. The files of the cartridge were then be transferred over to the computer's hard disk and uploaded to this repository.
